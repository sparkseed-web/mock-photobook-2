# 模写コーディング（フォトブック風サイト）

## 概要
学習目的で制作した模写コーディング作品です。

## 使用技術
- HTML
- CSS

## 学んだこと意識したこと
- コードを書き始める前にサイトの全体のレイアウトを分析しパーツごとに分け、メモをとることから始めました。
- デザインの共通点を見つけることでクラス名を付けやすくする。そうすることでコードの肥大化を防ぐことを学びました。
-  画像が並ぶセクションでは``display:grid;``を利用して画像を配置することでレスポンシブにも柔軟に対応できるようにしました。
